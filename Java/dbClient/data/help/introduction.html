Command Line Arguments

The general format supported to run the tool with the valid arguments:

DBGuiTool [-l logging_level [out_filename] [+]] [-m properties_filename] [-db db_controller] [-in filename]

Description of the command line valid arguments:

  • -l logging_level [out_filename] [+]

    Sets the logging level to the specified value from the logging root.

    • logging_level The logging level.
    • out_filename> The output verbose file.
    • + Output verbose to file and standard output.
  • -m properties_filename

    Set the property file to be used.

    • properties_filename The name of the property file.
  • -db db_controller

    Set the DB controller to be used.

    • db_controller The name of the DB controller, default riak.
  • -in filename

    Sets the input clients details file.

    • filename The input clients details filename.
